HOW TO RUN PROGRAM:

The program can be ran by hitting run in an IDE or through the command line.

compile the source files by using the javac command:
    command: javac Server.java
    command: javac Client.java

to run the classes use java command:
    command: java Server port_number 
    command: java Client ip_address port_number // after running this command the user will be prompted to enter message
